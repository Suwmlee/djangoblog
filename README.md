##Django blog

一个基于django开发的Blog程序。演示站点：http://suwmlee.duapp.com/

1.tinymce
2.内置评论
3.友情链接
4.主题
5.RSS

##依赖

1.数据库: MYSQL 5.6
2.django >=1.4

##安装步骤

1.创建数据库
2.执行python manage.py syncdb 同步数据库
3.python manage.py runserver 启动
4.浏览器访问 http://127.0.0.1:8000 ,后台地址 http://127.0.0.1:8000/e/admin/ ,口令在第二部创建
